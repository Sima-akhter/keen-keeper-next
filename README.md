# Keen Keeper Next

A modern task and productivity management web application built with Next.js. Users can organize tasks, track progress, visualize productivity data, and manage their workflow efficiently through a clean and responsive interface.

## 🚀 Features

- Create, update, and delete tasks
- Task status management
- Responsive user interface
- Dashboard with statistics
- Interactive charts using Recharts
- Toast notifications
- Modern UI with Tailwind CSS & DaisyUI
- Fast performance with Next.js App Router

## 🛠️ Technologies Used

### Frontend
- Next.js 16
- React 19
- React DOM
- Tailwind CSS 4
- DaisyUI

### Additional Libraries
- React Icons
- React Toastify
- Recharts

### Development Tools
- ESLint
- Babel React Compiler

## 📦 Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd keen-keeper-next
```

Install dependencies:

```bash
npm install
```

## ▶️ Running the Project

Start the development server:

```bash
npm run dev
```

Open:

```bash
http://localhost:3000
```

## 🏗️ Build for Production

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
